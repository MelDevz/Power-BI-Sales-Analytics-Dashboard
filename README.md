# Power BI Sales Dashboard 2026

This repository contains a personal Power BI project demonstrating end-to-end data analysis, modelling, and dashboard development using Power BI Desktop.

## Overview
The Sales Dashboard provides executive-level visibility into sales performance across time, products, and categories. It is designed to support business decision-making through clear KPIs, trend analysis, and interactive filtering.

### Key Insights Covered
- Total number of orders and customers
- Overall sales performance
- Sales trends by year and quarter
- Top-performing products
- Sales distribution by product category
- Interactive analysis by date range, country, and product category

## Tools & Technologies
- Power BI Desktop
- DAX (time intelligence, KPI measures)
- Data modelling (star schema concepts)
- Data visualisation best practices

## Dashboard Features
- KPI cards for high-level performance tracking
- Time-series analysis for sales trends
- Product-level performance ranking
- Category contribution analysis
- Slicers for dynamic filtering (date, country, product category)

## Business Value
This dashboard enables stakeholders to:
- Monitor sales performance at a glance
- Identify top-performing products and categories
- Analyse seasonal and quarterly sales trends
- Support data-driven decision-making through interactive exploration
